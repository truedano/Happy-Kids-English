# 🔑 API Key 設定指南

## 什麼是 API Key？

API Key 是您用來存取 Google Gemini AI 服務的金鑰。本應用程式需要 API Key 才能生成個性化的學習內容。

## 🎯 為什麼需要自行設定 API Key？

- ✅ **完全免費**：您使用自己的免費配額，無需支付任何費用
- ✅ **隱私保護**：您的金鑰只儲存在您的瀏覽器本機，不會傳送到任何伺服器
- ✅ **安全可靠**：程式碼中不包含任何敏感資訊
- ✅ **完全控制**：您可以隨時更新或刪除金鑰

## 📖 如何取得 API Key？

### 步驟 1：前往 Google AI Studio
點擊此連結：[https://aistudio.google.com/apikey](https://aistudio.google.com/apikey)

### 步驟 2：登入 Google 帳號
使用您的 Google 帳號登入（如果尚未登入）

### 步驟 3：建立 API Key
1. 點擊「**Create API Key**」按鈕
2. 選擇「**Create API key in new project**」
3. 等待幾秒鐘，系統會自動生成金鑰

### 步驟 4：複製金鑰
1. 點擊「**Copy**」按鈕複製金鑰
2. 金鑰格式類似：`AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`

### 步驟 5：貼到應用程式
1. 回到本應用程式
2. 在彈出的設定視窗中貼上金鑰
3. 點擊「**儲存並開始使用**」

## 🎉 完成！

設定完成後，您就可以開始使用 AI 功能生成學習內容了！

## ❓ 常見問題

### Q1: API Key 是免費的嗎？
**A:** 是的！Google Gemini API 提供免費配額：
- 每分鐘 15 次請求
- 每天 1,500 次請求
- 每月 100 萬 tokens

對於個人學習使用來說，這個配額非常充足。

### Q2: 我的 API Key 安全嗎？
**A:** 是的！您的 API Key：
- 只儲存在您的瀏覽器 localStorage 中
- 不會傳送到我們的伺服器（因為我們沒有伺服器！）
- 只有您自己可以看到和使用

### Q3: 如果清除瀏覽器資料會怎樣？
**A:** 清除瀏覽器資料（包括 localStorage）會刪除您儲存的 API Key。您需要重新輸入金鑰。

### Q4: 可以在多個裝置使用同一個 API Key 嗎？
**A:** 可以！您可以在不同裝置上使用同一個 API Key，只要不超過配額限制即可。

### Q5: 如何更新或刪除 API Key？
**A:** 點擊右上角的「⚙️」設定按鈕，即可更新或刪除金鑰。

### Q6: API Key 用完配額怎麼辦？
**A:** 您可以：
1. 等到隔天配額重置
2. 升級到付費方案（但通常免費配額就夠用了）
3. 建立新的 API Key

### Q7: 為什麼驗證失敗？
**A:** 可能的原因：
- 金鑰格式不正確（請確認已完整複製）
- 網路連線問題
- API 服務暫時無法使用
